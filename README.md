✏️ A blog based on GitHub Pages.

一个依赖 GitHub Pages，无需本地生成的静态博客

## Features

- Vue 2 / Vue-router / axios / vuex
- webpack 3 / Babel 6 / Stylus
- using [TravisCI](https://travis-ci.org)
- Hosting on GitHub Pages
- SessionStorage cache
